Components/Directives

  • app-select extends BoxBase
  • app-optgroup extends BoxBase
  • app-option extends BoxBase
  • [appOptionView]

Inputs

app-select

  • [placeholder]: string = ''
  • [canSelectNullableOptions]: boolean = false
  • [required]: boolean = false
  • [disabled]: boolean = false
  • [multiple]: boolean = false
  • [value]: T | T[] | null = null

app-optgroup

  • [label]!: string
  • [disabled]: boolean = false

app-option

  • [value]?: T
  • [selected]: boolean = false
  • [disabled]: boolean = false

Outputs

app-select

  • [change]: EventEmitter<AppSelectChange<T>>
  • [openedChange]: EventEmitter<boolean>

app-option

  • [value]?: T
  • [selectedChange]: EventEmitter<T>

Actions

app-select

  • toggle() - Abre o cierra la lista de opciones.
  • open() - Abre la lista de opciones.
  • close() - Cierra la lista de opciones.

app-option

  • select() - Emite el valor a seleccionar
