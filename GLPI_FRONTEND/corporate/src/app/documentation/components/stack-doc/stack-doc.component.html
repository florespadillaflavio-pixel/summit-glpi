Components/Directives

  • app-stack extends BoxBase

Inputs

  • [orientation]: 'vertical' | 'horizontal' | 'vertical-reverse' | 'horizontal-reverse' = 'vertical'
  • [wrap]: 'wrap' | 'nowrap' | 'wrap-reverse' = 'nowrap'
  • [gap]: string | number = '16px'
  • [noGap]: boolean = false
  • [justify]: 'center' | 'start' | 'end' | 'stretch' = 'stretch'
  • [align]: 'center' | 'start' | 'end' | 'stretch' = 'stretch'
