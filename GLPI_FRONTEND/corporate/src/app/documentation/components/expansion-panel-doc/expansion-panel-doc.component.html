Expansion Panel Module

<app-expansion-panel
    appearance="flat"
    expanded="false"
    disabled="false"
    (opened)="onOpened()"
    (closed)="onClosed()"
>
    <app-expansion-panel-header>
        Expansion Panel Header
    </app-expansion-panel-header>
    
    <app-stack padding>
        Content
    </app-stack>
</app-expansion-panel>

Components/Directives

  • app-expansion-panel
  • app-expansion-panel-header extends ListItemComponent

Inputs

app-expansion-panel
  • [appearance]: boolean = false
  • [expanded]: boolean = false
  • [disabled]: boolean = false

Outputs

app-expansion-panel
  • (opened): EventEmitter<void>
  • (closed): EventEmitter<void>

Actions

app-expansion-panel
  • open() - Opens panel
  • close() - Closes panel
  • toggle() - Toggles panel
