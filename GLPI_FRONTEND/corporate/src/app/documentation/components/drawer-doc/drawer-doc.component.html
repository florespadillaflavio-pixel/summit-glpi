Drawer Module

<app-drawer-container
    hasBackdrop="false"
    width="300" height="300"
    (backdropClick)="onBackdropClick()"
>
    <app-drawer #drawer
        mode="side"
        position="start"
        fixedInViewport="false"
        fixedTopGab="0"
        fixedBottomGap="0"
        opened="false"
        disableClose="false"
        (openedChange)="onOpenedChange($event)"
    >
        Drawer
        <button appFab (click)="drawer.close()">Close</button>
    </app-drawer>

    <app-drawer-content>
        Content
        <button appFab (click)="drawer.open()">Open</button>
        <button appFab (click)="drawer.toggle()">Toggle</button>
    </app-drawer-content>
</app-drawer-container>

Components/Directives

  • app-drawer-container, app-sidenav-container extends BoxBase
  • app-drawer, app-sidenav extends BoxBase
  • app-drawer-content, app-sidenav-content extends BoxBase

Inputs

app-drawer-container, app-sidenav-container
  • [hasBackdrop]: boolean = false

app-drawer, app-sidenav
  • [mode]: 'over' | 'side' = 'side'
  • [position]: 'start' | 'end' = 'start'
  • [fixedInViewport]: boolean = false
  • [fixedTopGap]: number = 0
  • [fixedBottomGap]: number = 0
  • [opened]: boolean = false
  • [disableClose]: boolean = false

Outputs

app-drawer-container, app-sidenav-container
  • (backdropClick): EventEmitter<void>

app-drawer, app-sidenav
  • (openedChange): EventEmitter<boolean>

Actions

app-drawer, app-sidenav
  • open() - Opens drawer
  • close() - Closes drawer
  • toggle() - Toggles drawer

Fixed Gaps

Fixed Top and Bottom Gaps Drawer 
<app-drawer-container hasBackdrop height="300" border>
    <app-drawer #drawer appStack padding borderRadius="0 16px 16px 0"
        mode="over"
        [fixedTopGap]="top.value"
        [fixedBottomGap]="bottom.value"
        [fixedInViewport]="fixedInViewport.checked"
    >
        Drawer
        <button appFab (click)="drawer.close()">Close</button>
    </app-drawer>

    <app-drawer-content appStack padding>
        Content
        <button appFab (click)="drawer.open()">Open</button>
        <button appFab (click)="drawer.toggle()">Toggle</button>
        <app-stack orientation="horizontal">
            <app-form-field>
                <app-label>Top</app-label>
                <button appPrefix appIconButton variant="tertiary" (click)="top.value = +top.value - 5"><app-icon>remove</app-icon></button>
                <input #top appInput appBoxBase textAlign="center" type="text" value="30">
                <button appSuffix appIconButton variant="tertiary" (click)="top.value = +top.value + 5"><app-icon>add</app-icon></button>
            </app-form-field>
            <app-form-field>
                <app-label>Bottom</app-label>
                <button appPrefix appIconButton variant="tertiary" (click)="bottom.value = +bottom.value - 5"><app-icon>remove</app-icon></button>
                <input #bottom appInput appBoxBase textAlign="center" type="text" value="30">
                <button appSuffix appIconButton variant="tertiary" (click)="bottom.value = +bottom.value + 5"><app-icon>add</app-icon></button>
            </app-form-field>
        </app-stack>
        <app-slide-toggle #fixedInViewport>Fixed in Viewport</app-slide-toggle>
    </app-drawer-content>
</app-drawer-container>
Drawer Content Top Bottom Fixed in Viewport info INFO: El '[fixedInViewport]' posiciona a 'app-drawer' fuera del contenedor en el padre relativo más próximo. Usar con precausión.
